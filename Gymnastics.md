# Πρόγραμμα Γυμναστικής - 3 Ημέρες (Push/Pull/Legs)

Αυτό είναι το τρέχον πρόγραμμα προπόνησης για τον Δημήτρη.

## Δομή Εβδομάδας
- **Ημέρα 1: Push** (Στήθος, Ώμοι, Τρικέφαλοι) - *Ολοκληρώθηκε*
- **Ημέρα 2: Pull** (Πλάτη, Δικέφαλοι) - *Προγραμματισμένο για αύριο*
- **Ημέρα 3: Legs** (Πόδια, Κοιλιακοί)

## Λεπτομέρειες Ασκήσεων

### Ημέρα 1: Push
- Πιέσεις στήθους (Bench Press)
- Πιέσεις ώμων (Overhead Press)
- Εκτάσεις τρικεφάλων

### Ημέρα 2: Pull
- Κωπηλατική (Rows)
- Έλξεις / Lat Pulldowns
- Κάμψεις δικεφάλων

### Ημέρα 3: Legs
- Squats (Καθίσματα)
- Leg Press
- Προβολές (Lunges)
- Κοιλιακοί / Plank
