# Hermes Memory

Hermes Memory is the built-in durable layer that helps Hermes remember Dimitris's preferences, stable setup facts, and recurring conventions across sessions.

It is good for personal/user context, for example:

- Dimitris prefers Greek/Greeklish, practical step-by-step guidance.
- Dimitris wants Hermes to act directly on the VPS when already there.
- Dimitris's setup uses VPS + Windows PC + Tailscale + local Ollama.

## Limitation

Hermes Memory should not become a dumping ground for every document, video, or research source. That broader source-backed knowledge belongs in the [[concepts/llm-wiki|LLM Wiki]].
