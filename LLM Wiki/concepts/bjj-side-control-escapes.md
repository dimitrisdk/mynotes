# BJJ Side Control Escapes & Reversals

Technique category for BJJ links where Dimitris is underneath side control and the goal is to recover guard, create frames, reverse, turtle, wrestle up, or otherwise escape.

## Include here

- side control escapes
- side control reversals
- frames from bottom side control
- elbow-knee escape from side control
- underhook escapes / wrestle-ups from bottom side control
- ghost escape style movements

## Do not include here

- top player attacks from side control → use `bjj/side-control-attacks`
- closed guard attacks/sweeps → use `bjj/closed-guard`
- mount escapes → use `bjj/mount-escapes`
- guard passing → use `bjj/guard-passing`
