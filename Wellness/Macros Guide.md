# Πρότυπο Γευμάτων — 2200 kcal / 170g Πρωτεΐνη

Σκοπός: πρότυπο για να ξέρεις πώς καταλήγεις στα macros χωρίς να μετράς κάθε μέρα κάθε γραμάριο.

## Macros goals
- Θερμίδες: **2200 kcal**
- Πρωτεΐνη: **170g**
- Λιπαρά: ~65–80g
- Υδατάνθρακες: ~240–280g

## Πρότυπο για 4 γεύματα
| Γεύμα | Κύριο | Θερμίδες | Πρωτεΐνη | Λιπαρά | Υδατάνθρακες |
| :--- | :--- | ---: | ---: | ---: | ---: |
| Πρωινό | Παράδειγμα: γιαούρτι 2% + protein powder + μπανάνα + κανέλα | ~550 kcal | ~45g | ~10g | ~60g |
| Μεσημεριανό | Παράδειγμα: ρύζι + κοτόπουλο + λαχανικά + λίγο λάδι | ~600 kcal | ~50g | ~15g | ~70g |
| Σνακ | Παράδειγμα: protein ice cream ή protein bar + φρούτο | ~350 kcal | ~30g | ~5g | ~40g |
| Βραδινό | Παράδειγμα: wrap πλήρους αλεύρι + κοτόπουλο + τυρί + σαλάτα | ~700 kcal | ~45g | ~15g | ~50g |
| **Σύνολο** | | **~2200** | **~170g** | **~45g** | **~220g** |

## Ευέλικτο πρότυπο (ανάλογα με την ημέρα)
- Πρωινό: **πάντα υψηλή πρωτεΐνη** για saturation
- Μεσημεριανό: **πρωτεΐνη + υδατάνθρακες** για ενέργεια / work
- Σνακ: **πρωτεΐνη + λίγο υδατάνθρακες** για να μην πεινάσεις
- Βραδινό: **πρωτεΐνη + λίγα υδατάνθρακες + λιπαρά** για recovery και για να μην πεινάσεις το βράδυ

## Ανάλογα προς την προπόνηση
- **Γυμναστήριο / BJJ / Yoga**: πρόσθεσε +40–80g υδατάνθρακες (π.χ. ρύζι, πατάτες) πριν ή μετά
- **Αδρανής ημέρα**: μείωσε υδατάνθρακες κατά ~40–60g (προτίμησε περισσότερη σαλάτα + λιπαρά)

## Ανέκδοτα για τυχαία γεύματα
| Τυχαίο γεύμα | Πότε |
| :--- | :--- |
| Protein Ice Cream | Όποτε θες γλυκό — υπολογίζεται ως σνακ |
| Souvlaki / gyros | Μια φορά την εβδομάδα OK, αλλά μείωσε υδατάνθρακες αλλού |
| Skroutz order / delivery | Προσάρμοσε τα υπόλοιπα γεύματα χωρίς να πειράζεις πολύ |
| Καφές με γάλα / latte | Προσθέτει ~30–80 kcal ανάλογα με το γάλα |
| Shot Party / λίγο αλκοόλ | Μείωσε λιπαρά ή υδατάνθρακες αλλού |

## Ενδεικτικά προϊόντα για γρήγορη επιλογή
- **Πρωτεΐνη**: protein powder (20–25g ανά σκεύος), τυρί cottage/ricotta (7–10g/50g), γιαούρτι 2% (10g/200g)
- **Υδατάνθρακες**: ολικής αλέσεως πείτα/ρύζι/ζυμαρικά, πατάτες, μπανάνα, μέλι λίγο
- **Λιπαρά**: ελαιόλαδο, αβοκάντο, φουντούκια, φύλλα λάχανου
- **Vegetables**: όλα τα λαχανικά ελεύθερα — δεν μετρούνται σκληρά για macros
