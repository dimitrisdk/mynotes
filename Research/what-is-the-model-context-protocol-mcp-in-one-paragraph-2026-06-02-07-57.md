# What is the Model Context Protocol (MCP) in one paragraph?

Το έρευνα αποθηκεύτηκε στο **Obsidian Vault** (`MCP-Research.md`).

**Σύνοψη:** Δεν υπήρχε προηγούμενη γνώση για MCP στο vault. Βρήκα ότι το MCP είναι open-source standard της Anthropic που λειτουργεί σαν «USB-C για AI» — ενώνει LLMs με εξωτερικά tools/databases μέσω client-server αρχιτεκτονικής (JSON-RPC 2.0). Λύνει το N×M integration πρόβλημα. Το Hermes Agent το υποστηρίζει natively με native MCP client. Η πλήρης ανάλυση με πηγές είναι παραπάνω και στο vault.
